# Fake Profile Detection Using Machine Learning

## Project Overview
This project is developed to detect fake social media profiles using machine learning techniques. The system analyzes profile-related features and predicts whether a profile is genuine or fake.

## Objective
To identify fake social media accounts using machine learning algorithms for improving online security and reducing fraudulent activities.

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Algorithms Used
- Logistic Regression
- Support Vector Machine (SVM)
- Random Forest
- Gradient Boosting

## Workflow
1. Data Collection
2. Data Cleaning
3. Feature Engineering
4. Exploratory Data Analysis (EDA)
5. Data Preprocessing
6. Model Training
7. Model Evaluation
8. Hyperparameter Tuning
9. Prediction System

## Dataset
The dataset contains social media profile information used for training and testing the machine learning models.

## Results
The models were evaluated using accuracy score, confusion matrix, and classification report. Gradient Boosting achieved the best performance among the implemented algorithms.
